# Decisions

**2026-06-23**

- Background: playground 작업에서 문서 작성 실패 패턴을 조사했고, 실제 개선 대상이 `structure-first-docs`에 있는 것으로 좁혀졌다.
- Decision: 새 스킬이나 `AGENTS.md` 보강이 아니라 `structure-first-docs`의 `Compose Rules`에 짧은 보강을 넣는 방향으로 작업을 시작한다.
- Why: 기존 스킬이 이미 문서 의미 구조, 소유 문서, 중복, 격하를 맡고 있어 새 소유 영역을 만들 필요가 약하다.
- Impact: 다음 세션은 `SKILL.md`와 `SKILL.ko.md`의 의미 동기화에 집중하고, playground 조사 문서를 다시 확장하지 않는다.
